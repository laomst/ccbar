# CCBar

一个 IntelliJ IDEA 插件，在工具栏中添加可配置的快捷按钮，用于快速启动命令行工具（如 AI 编程助手）。

## 功能特性

- **多快捷按钮**：支持在工具栏添加多个按钮，每个按钮代表一个命令类别
- **直接命令模式**：Button 可绑定直接命令，点击后一键执行（不弹出菜单）
- **命令预览**：弹出菜单中实时预览将要执行的命令
- **灵活配置**：支持配置基础命令、参数变体、工作目录等
- **终端集成**：自动在终端中执行命令，支持自定义终端名称
- **编辑器模式**：终端可在编辑器区域以 Tab 形式打开，适合长时间运行的 AI 编程助手
- **项目级配置**：支持为每个项目设置独立的按钮配置，配置跟随项目

## 使用方式

### 直接命令模式

如果 Button 配置了直接命令，点击后直接执行：

1. 点击工具栏按钮（如 `NPM Test`）
2. 弹出终端命名对话框
3. 确认后直接执行命令

### 选项列表模式

如果 Button 未配置直接命令，点击后弹出选项菜单：

1. 点击工具栏上的按钮（如 `🤖 Claude`）
2. 弹出选项菜单，每行显示：
   - **选项名称**：点击执行基础命令
   - **命令预览输入框**：点击执行基础命令；悬浮子按钮时显示完整命令
   - **子按钮列表**：点击执行带参数的完整命令

```
┌──────────────────────────────────────────────────────────────────┐
│ Model     │ claude                    │ [Default][Sonnet][Opus] │
│ Workspace │ claude                    │ [Home][Work]            │
│ System    │ claude                    │ [Dev]                   │
└──────────────────────────────────────────────────────────────────┘
```

### 命令生成规则

**直接命令模式**：
```
最终执行命令 = Button.command
```

**选项列表模式**：
```
最终执行命令 = Option.baseCommand + (SubButton.params 不为空 ? " " + SubButton.params : "")
```

| 模式 | 点击目标 | 执行命令 |
|------|----------|----------|
| 直接命令 | Button | `npm test` |
| 选项列表 | 选项名称 | `claude` |
| 选项列表 | 子按钮 [Sonnet] | `claude --model sonnet` |

## 配置

进入 **Settings → Tools → CCBar** 进行配置：

- 添加/编辑/删除工具栏按钮
- **直接命令模式**：为 Button 配置 command、工作目录、终端名称
- **选项列表模式**：为 Button 配置选项（Option）和子按钮（SubButton）
- 支持导入/导出 JSON 格式配置

### 系统配置与项目配置

CCBar 支持两级配置：

| 配置级别 | 存储位置 | 作用域 |
|---------|---------|--------|
| 系统配置 | IDEA 全局配置目录 | 所有项目共享 |
| 项目配置 | `.idea/ccbar.xml` | 仅当前项目 |

**使用场景**：
- **系统配置**：适合通用的命令配置，所有项目共享
- **项目配置**：适合项目特定的命令，配置跟随项目，可被版本控制

**配置优先级**：
1. 如果项目配置已启用 → 使用项目配置
2. 如果项目配置未启用/不存在 → 使用系统配置

在设置页面中可以：
- 点击"启用项目配置"将系统配置复制到项目配置
- 点击"禁用项目配置"停用项目配置（不删除数据）
- 点击"重置为系统配置"将项目配置重置为当前系统配置的副本

## 构建

```bash
./gradlew build              # 构建插件
./gradlew runIde             # 启动沙箱 IDEA 实例并加载插件
./gradlew buildPlugin        # 构建可分发的插件 ZIP 包
```
